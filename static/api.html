TaskTray API

GET/api/items

Return all non-hidden dashboard items.

POST/api/items

Create a new manual item. Body: {"title": "...", "status": "active", "priority": "p1", "notes": "..."}

PATCH/api/items/:id

Update fields on an existing item.

DELETE/api/items/:id

Hide/delete an item from the dashboard.

GET/api/stats

Return aggregate stats (total, by_source, by_status).

POST/api/sync

Trigger an immediate disk + Obsidian sync.

GET/api/health

Health check: uptime, last sync time, item count.

GET/api/config

Return sanitized configuration (scan_dirs, vault_path, categories).

GET/api/export?format=json|csv

Export all non-hidden items as a JSON or CSV download. Defaults to JSON.
